# Wall Calendar

Interactive wall calendar built with React and Vite for the internship challenge.

## Structure

```text
frontend/
  index.html
  package.json
  src/
    App.jsx
    main.jsx
    index.css
    components/
      calendar/
        CalendarGrid.jsx
        HeroArtwork.jsx
        NotesPanel.jsx
        WallCalendar.jsx
        wall-calendar.css
    utils/
      calendar.js
```

## Included features

- Wall-calendar inspired hero section
- Month navigation
- Start and end date range selection
- Month memo and range-based notes
- Responsive desktop and mobile layout
- Local persistence with `localStorage`

## Run locally

```bash
cd frontend
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.
